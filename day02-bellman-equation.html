Day 2：贝尔曼方程

Bellman Equation — 把无限求和变成一步递推

1. 核心递推

回报递推（贝尔曼方程之源）
$G_t = R_{t+1} + \gamma G_{t+1}$
💡 核心洞察

今天的总回报 = 今天的奖励 + 折现后的明天总回报。贝尔曼方程把"无限求和"变成了"一步递推"。

2. 贝尔曼期望方程

Vπ 贝尔曼期望方程
$V^\pi(s) = \sum_{a} \pi(a \mid s) \sum_{s'} P(s' \mid s, a) \Big[ R(s, a, s') + \gamma V^\pi(s') \Big]$
Qπ 贝尔曼期望方程
$Q^\pi(s, a) = \sum_{s'} P(s' \mid s, a) \Big[ R(s, a, s') + \gamma V^\pi(s') \Big]$

给定策略 $\pi$，评估该策略的价值——回答"按 $\pi$ 走，值多少钱？"

3. 贝尔曼最优方程

V* 贝尔曼最优方程
$V^*(s) = \max_{a} \sum_{s'} P(s' \mid s, a) \Big[ R(s, a, s') + \gamma V^*(s') \Big]$
Q* 贝尔曼最优方程
$Q^*(s, a) = \sum_{s'} P(s' \mid s, a) \Big[ R(s, a, s') + \gamma \max_{a'} Q^*(s', a') \Big]$

直接找最优——把 $\sum \pi$ 换成 $\max$。

4. 期望方程 vs 最优方程

贝尔曼期望方程贝尔曼最优方程
前提给定策略 $\pi$最优策略 $\pi^*$
运算$\sum_a \pi(a|s)$ 加权平均$\max_a$ 取最大值
问题评估策略好坏寻找最优策略
对应算法策略评估价值迭代 / Q-Learning

5. V 与 Q 的关系全景

Vπ→ 加权平均 →Qπ→ 转移期望 →Vπ(s')
等式公式含义
① V→Q$V^\pi(s) = \sum_a \pi(a|s) Q^\pi(s,a)$Q 的加权平均
② Q→V$Q^\pi(s,a) = \sum_{s'} P[R + \gamma V^\pi(s')]$一步后的 V
③ 组合$V^\pi(s) = \sum_a \pi \sum_{s'} P[R + \gamma V^\pi(s')]$期望方程
④ 最优$V^*(s) = \max_a \sum_{s'} P[R + \gamma V^*(s')]$最优方程
从 ③ 到 ④ 只改了一个符号：$\sum\pi$ → $\max$。含义从"评估策略"变成"寻找最优策略"。

6. 手算示例：1×3 网格

[ S ]──[ A ]──[ G ]    γ=0.9, 每步 R=-1, 到 G 奖励 +10

从后往前推:
  V(A) = max(Q(A,←), Q(A,→)) = max(-1+0.9V(S), 10+0) = max(6.2, 10) = 10
  V(S) = -1 + 0.9 × V(A) = -1 + 9 = 8

结果: V*(S)=8, V*(A)=10, π*(S)=→, π*(A)=→

7. 核心总结

1回报递推
$G_t = R_{t+1} + \gamma G_{t+1}$
2期望方程
评估策略：$\sum\pi$ 加权平均
3最优方程
找最优策略：$\max$ 取最大
4核心直觉
当前价值 = 即时奖励 + 折现未来价值
