Day 5：时序差分学习

Temporal Difference — MC 和 DP 的完美融合

1. TD(0)：最优雅的公式

TD(0) 更新规则
$V(S_t) \leftarrow V(S_t) + \alpha \big[ R_{t+1} + \gamma V(S_{t+1}) - V(S_t) \big]$
TD 误差
$\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$
💡 TD = MC + DP

像 MC 一样不需要模型（从经验学）
像 DP 一样可以自举（用 V(s') 代替 G）
每步更新，不需要等回合结束

2. 三种更新目标对比

方法更新目标需要模型？等结局？自举？
DP$R + \gamma V(s')$✅❌✅
MC$G_t$（完整回报）❌✅❌
TD$R + \gamma V(s')$❌❌✅

3. 偏差-方差权衡

MCTD(0)
偏差无偏有偏（V(s') 不准）
方差高（G 累积全程随机性）低（只依赖一步）
收敛速度慢快
MC：看完电影再评价——准确但每次感受不同。TD：看预告片猜评分——可能猜错但很稳定。

4. MC 误差 = TD 误差的折扣和

关键恒等式
$G_t - V(S_t) = \delta_t + \gamma\delta_{t+1} + \gamma^2\delta_{t+2} + \cdots + \gamma^{T-t-1}\delta_{T-1}$

MC 误差可以分解为所有后续 TD 误差的折扣和。批量更新时，TD 和 MC 可能收敛到不同结果——TD 利用了马尔可夫结构。

5. 收敛性

收敛条件
$\sum_{t=1}^{\infty} \alpha_t = \infty, \quad \sum_{t=1}^{\infty} \alpha_t^2 < \infty$

步长无穷大保证能走到目标，步长平方和有限保证最终稳定。

6. TD 的历史地位

🏆 RL 中最重要的思想之一

TD 首次实现了：① 不需要模型 ② 不需要等回合结束 ③ 在线、增量、每步更新
后续 SARSA、Q-Learning、DQN、Actor-Critic 全部建立在 TD 之上。

7. 核心总结

1TD(0) 更新
$V \leftarrow V + \alpha[R + \gamma V' - V]$
2TD 误差
$\delta = R + \gamma V' - V$
3TD = MC + DP
无模型 + 自举
4偏差-方差
有偏低方差 vs 无偏高方差
