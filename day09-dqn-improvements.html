问题原因后果解决方案
Q 值过估计max 操作引入正偏差策略次优Double DQN

2. Double DQN：消除过估计

问题：max 的正偏差

DQN 目标（过估计风险）
$y = r + \gamma \max_{a'} Q(s', a'; \theta^-)$

max 操作从噪声估计中选最大值 → 系统性偏高。

解决：选动作和评估分离

Double DQN 目标
$y = r + \gamma Q\Big(s', \underbrace{\arg\max_{a'} Q(s', a'; \theta)}_{\text{主网络选动作}}; \underbrace{\theta^-}_{\text{目标网络评估}}\Big)$
选动作评估动作
DQN目标网络目标网络（同一个）
Double DQN主网络目标网络（分离）

手算示例

3 个动作, 目标网络 Q(s',·;θ⁻) = [3, 5, 4], 主网络 Q(s',·;θ) = [3.2, 4.8, 4.5]

DQN:        max Q(s',a';θ⁻) = 5        → y = r + γ·5
Double DQN: argmax Q(s',a';θ) = a₂(4.8) → y = r + γ·Q(s',a₂;θ⁻) = r + γ·5

看起来一样? 不一定! 如果主网络 argmax 选了不同动作:
  Q(s',·;θ) = [3.2, 4.5, 4.8] → argmax = a₃ → Q(s',a₃;θ⁻) = 4 ≠ 5
  Double DQN 给出更保守的估计!

3. Dueling DQN：分离 V 和 A

Q 值分解
$Q(s, a) = \underbrace{V(s)}_{\text{状态价值}} + \underbrace{A(s, a)}_{\text{优势函数}}$
Dueling 组合公式
$\hat{Q}(s,a) = V(s;\beta) + \Big(A(s,a;\alpha) - \frac{1}{|\mathcal{A}|}\sum_{a'} A(s,a';\alpha)\Big)$
输入 s→共享层→V 流 + A 流→Q = V+A-mean(A)
为什么减去 mean(A)?

直接 Q=V+A 有可辨识性问题（V 和 A 可同时加减常数）。强制优势均值为 0，使 V 等于最优 Q 值。

4. PER：优先经验回放

优先级定义
$p_i = |\delta_i| + \epsilon \quad \text{(TD 误差越大, 优先级越高)}$
采样概率
$P(i) = \frac{p_i^\alpha}{\sum_k p_k^\alpha} \quad (\alpha=0: \text{均匀}, \alpha=1: \text{全优先级})$
重要性采样权重
$w_i = \left(\frac{1}{N} \cdot \frac{1}{P(i)}\right)^\beta \quad (\beta: 0\to\text{不修正}, 1\to\text{完全修正})$

手算示例

3 条经验, δ=[0.1, 0.5, 2.0], ε=0.01, α=0.6, β=0.4

优先级: p = [0.11, 0.51, 2.01]
采样概率: P ≈ [0.097, 0.269, 0.634]  ← 经验3被采概率最高
IS权重 (β=0.4): w₃ = (1/3 · 1/0.634)^0.4 ≈ 0.777  ← 高优先级权重 < 1
