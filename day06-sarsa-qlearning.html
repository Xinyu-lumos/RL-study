Day 6：SARSA 与 Q-Learning

TD 控制 — 从评估策略到寻找最优策略

1. 回顾：从 V 到 Q 的必然性

Day 5 的 TD(0) 只评估 V。但无模型时想改进策略必须知道 Q：

DP（有模型）
$\pi'(s) = \arg\max_a \sum_{s'} P(s'|s,a)[R + \gamma V(s')]$ — 需要 $P$
无模型
$\pi'(s) = \arg\max_a Q(s,a)$ — 不需要 $P$，直接比大小

2. SARSA On-Policy

SARSA 更新规则
$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \big[ R_{t+1} + \gamma Q(S_{t+1}, A_{t+1}) - Q(S_t, A_t) \big]$

名字来自五元组：St → At → Rt+1 → St+1 → At+1

S, A→执行 A, 得 R, S'→ε-贪心选 A'→用 Q(S',A') 更新
On-Policy = 所见即所学

采样用的 A'（ε-贪心）和更新目标里的 A' 来自同一个策略。SARSA 知道自己会探索犯错，所以学到的策略会规避风险。

3. Q-Learning Off-Policy

Q-Learning 更新规则
$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \big[ R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t) \big]$

和 SARSA 只有一处不同：$\max_{a'}$ 代替了 $Q(S',A')$

TD 目标含义
SARSA$R + \gamma Q(S', A')$用策略实际选的 $A'$
Q-Learning$R + \gamma \max_{a'} Q(S', a')$用最优的 $a'$
Off-Policy = 两套策略

行为策略（ε-贪心）：负责探索采样
目标策略（纯贪心）：我们真正想得到的
Q-Learning 直接学目标策略的 $Q^*$，且不需要重要性采样。

4. Expected SARSA 低方差

Expected SARSA
$Q(S,A) \leftarrow Q(S,A) + \alpha \big[ R + \gamma \sum_{a'} \pi(a'|S') Q(S',a') - Q(S,A) \big]$

不采样 $A'$，直接对策略取期望 → 消除采样方差。

当 $\pi$ 是纯贪心时，Expected SARSA 退化为 Q-Learning。

5. Cliff Walking 实验

S ── C C C C C C C C C C ── G    (S=起点 G=终点 C=悬崖-100)

         上方绕行（安全路径）
S · · · · · · · · · · · G
         悬崖边缘（最短路径）
贪心路径训练奖励原因
SARSA16步（绕行）-86考虑探索会掉崖，保守绕行
Q-Learning12步（贴崖）-149假设完美执行，ε-贪心频繁掉崖
Expected SARSA14步-73降低方差，综合最优
Q-Learning 学到"理论上最美"的策略，SARSA 学到"实际上最能活"的策略。

6. 三种算法总对比

算法更新公式类型方差风险
SARSA$Q + \alpha[R+\gamma Q(S',A')-Q]$On-Policy高保守
Expected SARSA$Q + \alpha[R+\gamma\sum\pi Q - Q]$均可中中间
Q-Learning$Q + \alpha[R+\gamma\max Q - Q]$Off-Policy低激进

7. 算法选择指南

1环境有风险？
探索时失误代价大 → SARSA
2追求最优？
能容忍探索失误 → Q-Learning
3想二者兼顾？
牺牲一点计算换低方差 → Expected SARSA
4注意过估计
Q-Learning 的 max 引入正偏差 → Day 9 修复

8. 核心总结

#概念一句话

2Q-Learning用 max 更新 → 激进最优
3唯一区别$Q(S',A')$ vs $\max Q(S',a')$
