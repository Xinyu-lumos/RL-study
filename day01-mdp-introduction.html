Day 1：引言与马尔可夫决策过程

Reinforcement Learning — MDP: 问题建模的数学框架

1. 什么是强化学习

强化学习研究智能体（Agent）如何在与环境（Environment）的交互中，通过试错学习最优行为策略，以最大化累积奖励。

范式数据反馈目标
监督学习带标签数据集正确答案拟合映射
无监督学习无标签数据无显式反馈发现结构
强化学习交互序列奖励信号最大化累积奖励

三大特征：没有标准答案、延迟反馈、序列决策。

2. Agent-Environment 交互

要素符号说明
智能体—学习者和决策者
环境—Agent 交互的外部世界
状态$S_t \in \mathcal{S}$$t$ 时刻环境快照
动作$A_t \in \mathcal{A}$$t$ 时刻 Agent 的选择
奖励$R_{t+1} \in \mathbb{R}$执行动作后的反馈信号
Agent→ 动作 At →Environment→ St+1, Rt+1 →Agent

产生的轨迹：$S_0, A_0, R_1, S_1, A_1, R_2, \ldots$

3. 马尔可夫决策过程（MDP）

马尔可夫性质

核心假设
$P(S_{t+1} \mid S_t, A_t, \ldots, S_0) = P(S_{t+1} \mid S_t, A_t)$
未来只取决于现在，与过去无关。

MDP 五元组

MDP 定义
$(\mathcal{S}, \mathcal{A}, P, R, \gamma)$
符号名称含义
$\mathcal{S}$状态空间所有可能状态的集合
$\mathcal{A}$动作空间所有可能动作的集合
$P(s'|s,a)$转移概率执行 $a$ 后到达 $s'$ 的概率
$R(s,a)$奖励函数$\mathbb{E}[R_{t+1} \mid S_t=s, A_t=a]$
$\gamma \in [0,1]$折扣因子权衡即时与远期奖励

转移概率

状态转移概率
$P(s' \mid s, a) = P(S_{t+1}=s' \mid S_t=s, A_t=a)$

性质：$P(s'|s,a) \geq 0$，$\sum_{s'} P(s'|s,a) = 1$（归一化）

折扣因子 γ

γ 值行为场景
$\gamma = 0$只关心即时奖励短视
$\gamma = 0.9$平衡即时与远期常用
$\gamma \to 1$同样关心远期长序列任务

4. 回报与价值函数

回报 Gt

折扣累积回报
$G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$
关键递推
$G_t = R_{t+1} + \gamma G_{t+1}$

状态价值函数 Vπ

V 的定义
$V^\pi(s) = \mathbb{E}_\pi[G_t \mid S_t = s] = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \;\Big|\; S_t = s\right]$

动作价值函数 Qπ

Q 的定义
$Q^\pi(s, a) = \mathbb{E}_\pi[G_t \mid S_t = s, A_t = a]$

V 与 Q 的关系

加权平均
$V^\pi(s) = \sum_{a} \pi(a \mid s) Q^\pi(s, a)$

5. 策略（Policy）

策略定义
$\pi(a \mid s) = P(A_t = a \mid S_t = s)$
最优策略
$V^*(s) = \max_\pi V^\pi(s), \quad Q^*(s,a) = \max_\pi Q^\pi(s,a)$
从 Q* 提取最优策略
$\pi^*(s) = \arg\max_a Q^*(s,a)$

6. Grid World 实例

┌─────┬─────┬─────┐
│  S  │     │  T  │   S = 起点 (0,0)
├─────┼─────┼─────┤   G = 终点 (2,2), R=+10
│     │  #  │     │   T = 陷阱 (0,2), R=-10
├─────┼─────┼─────┤   # = 障碍 (1,1)
│     │     │  G  │
└─────┴─────┴─────┘   γ=0.9, 其他每步 R=-1
S (0,0)→→(0,1)→↓(1,2)→↓G (2,2) +10

7. 核心总结

1马尔可夫性质
$P(S_{t+1}|S_t,A_t)$ 未来只跟现在有关
2MDP
$(\mathcal{S},\mathcal{A},P,R,\gamma)$
3回报
$G_t = \sum \gamma^k R_{t+k+1}$
4V 和 Q
状态价值 vs 动作价值
5策略
$\pi(a|s)$ 决策规则
6RL 目标
找到 $\pi^*$ 最大化 $V^\pi$
