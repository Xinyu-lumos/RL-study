Day 1 引言与马尔可夫决策过程

什么是强化学习

强化学习研究智能体（Agent）如何在与环境（Environment）的交互中，通过试错学习最优行为策略，以最大化累积奖励。

范式数据反馈目标
监督学习带标签数据集正确答案拟合映射
无监督学习无标签数据无显式反馈发现结构
强化学习交互序列奖励信号最大化累积奖励

Agent-Environment 交互循环

Agent → 动作 At → Environment → St+1, Rt+1 → Agent

MDP 五元组

MDP 定义
$(\mathcal{S}, \mathcal{A}, P, R, \gamma)$ — 状态空间、动作空间、转移概率、奖励函数、折扣因子

马尔可夫性质

核心假设
$P(S_{t+1} \mid S_t, A_t, \ldots, S_0) = P(S_{t+1} \mid S_t, A_t)$
未来只取决于现在，与过去无关。当前状态包含了所有用于决策的历史信息。

回报与价值函数

回报 Gt
$G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$
状态价值函数 Vπ
$V^\pi(s) = \mathbb{E}_\pi[G_t \mid S_t = s]$
动作价值函数 Qπ
$Q^\pi(s, a) = \mathbb{E}_\pi[G_t \mid S_t = s, A_t = a]$

Day 1 核心总结

1马尔可夫性质
$P(S_{t+1}|S_t,A_t)$ 未来只跟现在有关
2MDP
$(\mathcal{S},\mathcal{A},P,R,\gamma)$
3回报
$G_t = \sum \gamma^k R_{t+k+1}$
4V 和 Q
$V^\pi = \mathbb{E}[G|s]$, $Q^\pi = \mathbb{E}[G|s,a]$

Day 2 贝尔曼方程

核心递推：从无限和到一步

回报递推（贝尔曼方程之源）
$G_t = R_{t+1} + \gamma G_{t+1}$
💡 核心洞察
贝尔曼方程把"无限求和"变成了"一步递推"：今天的总资产 = 今天的收入 + 折现后的明天的总资产。

贝尔曼期望方程

Vπ 的贝尔曼期望方程
$V^\pi(s) = \sum_{a} \pi(a \mid s) \sum_{s'} P(s' \mid s, a) \Big[ R(s, a, s') + \gamma V^\pi(s') \Big]$
Qπ 的贝尔曼期望方程
$Q^\pi(s, a) = \sum_{s'} P(s' \mid s, a) \Big[ R(s, a, s') + \gamma V^\pi(s') \Big]$

贝尔曼最优方程

V* 的贝尔曼最优方程
$V^*(s) = \max_{a} \sum_{s'} P(s' \mid s, a) \Big[ R(s, a, s') + \gamma V^*(s') \Big]$
Q* 的贝尔曼最优方程
$Q^*(s, a) = \sum_{s'} P(s' \mid s, a) \Big[ R(s, a, s') + \gamma \max_{a'} Q^*(s', a') \Big]$

期望方程 vs 最优方程

贝尔曼期望方程贝尔曼最优方程
前提给定策略 π最优策略 π*
运算$\sum_a \pi(a|s)$ 加权平均$\max_a$ 取最大值
问题评估策略寻找最优策略
对应算法策略评估价值迭代 / Q-Learning

V 与 Q 关系图

Vπ → 加权平均 → Qπ → 转移期望 → Vπ(s')

Day 3 动态规划

策略评估

迭代更新
$V_{k+1}(s) = \sum_a \pi(a \mid s) \sum_{s'} P(s'|s,a) [R + \gamma V_k(s')]$
V=0 → 期望方程更新 → Δ<θ ? → 输出 Vπ

策略迭代

贪心改进
$\pi'(s) = \arg\max_a \sum_{s'} P(s'|s,a) [R + \gamma V^\pi(s')]$
π → 评估 → Vπ → 贪心 → π' → 评估 → ...

价值迭代

V 的贝尔曼最优更新
$V_{k+1}(s) = \max_a \sum_{s'} P(s'|s,a) [R + \gamma V_k(s')]$

三种 DP 算法对比

策略评估策略迭代价值迭代
目的评估 π找 π*找 π*
公式$\sum\pi\sum P[R+\gamma V]$交替评估+改进$\max_a\sum P[R+\gamma V]$
评估深度到收敛到收敛一步

广义策略迭代（GPI）

🔑 统一框架
几乎所有 RL 算法都是 GPI 的实例：评估（E）和改进（I）两个过程交替推进。E 和 I 不需要做到极致——只要两者不断推进，系统就会向最优收敛。

Day 4 蒙特卡洛方法

核心思想：用采样替代模型

MC 估计
$V^\pi(s) \approx \frac{1}{N(s)} \sum_{i=1}^{N(s)} G_t^{(i)}$
想知道骰子抛出 6 的概率？不用推导物理公式，掷 10000 次然后数。

增量更新

增量平均（为 TD 铺垫）
$V(S_t) \leftarrow V(S_t) + \alpha \big(G_t - V(S_t)\big)$

ε-贪心策略

探索与利用的平衡
$\pi(a|s) = \begin{cases} 1 - \varepsilon + \frac{\varepsilon}{|\mathcal{A}|} & a = \arg\max Q \\ \frac{\varepsilon}{|\mathcal{A}|} & \text{其他} \end{cases}$

On-Policy vs Off-Policy

On-PolicyOff-Policy
采样策略就是目标策略与目标策略不同
最终策略ε-最优纯贪心最优
方差低高（重要性采样）

MC vs DP

+MC 优势
不需要模型、无偏、状态独立
−MC 劣势
必须等回合结束、高方差、连续任务不可用

Day 5 时序差分学习

TD(0)：最优雅的公式

TD(0) 更新规则
$V(S_t) \leftarrow V(S_t) + \alpha \big[ R_{t+1} + \gamma V(S_{t+1}) - V(S_t) \big]$
TD 误差
$\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$
💡 TD = MC + DP

像 MC 一样不需要模型（从经验学）
像 DP 一样可以自举（用 V(s') 代替 G）
每步更新，不需要等回合结束

三种更新目标对比

方法更新目标需要模型？需要等结局？自举？
DP$R + \gamma V(s')$✅❌✅
MC$G_t$❌✅❌
TD$R + \gamma V(s')$❌❌✅

偏差-方差权衡

MCTD(0)
偏差无偏有偏（V(s') 不准）
方差高（G 累积全程随机性）低（只依赖一步）
收敛速度慢快

MC 误差 = TD 误差的折扣和

关键恒等式
$G_t - V(S_t) = \delta_t + \gamma\delta_{t+1} + \gamma^2\delta_{t+2} + \cdots + \gamma^{T-t-1}\delta_{T-1}$

Day 6 SARSA 与 Q-Learning

SARSA：On-Policy TD 控制

SARSA 更新 On-Policy
$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \big[ R_{t+1} + \gamma Q(S_{t+1}, A_{t+1}) - Q(S_t, A_t) \big]$

名字来自五元组：S₀, A₀, R₁, S₁, A₁。TD 目标中使用的 $A_{t+1}$ 是同一策略 ε-贪心选的。

Q-Learning：Off-Policy TD 控制

Q-Learning 更新 Off-Policy
$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \big[ R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t) \big]$

TD 目标使用 $\max$ 而非策略选择的动作——直接评估最优价值函数。

Expected SARSA

Expected SARSA
$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \big[ R_{t+1} + \gamma \sum_{a'} \pi(a'|S_{t+1}) Q(S_{t+1}, a') - Q(S_t, A_t) \big]$

三种算法核心对比

SARSAExpected SARSAQ-Learning
类型On-Policy可 On 可 OffOff-Policy
TD 目标$Q(S', A')$$\sum \pi Q$$\max Q$
方差最高中最低
风险偏好保守中间激进

Cliff Walking 实验

Cliff Walking 关键发现

SARSA 学到上方绕行的安全路径（考虑 ε-贪心会掉崖的风险）
Q-Learning 学到悬崖边缘的最短路径（假设执行最优策略）
但训练时 Q-Learning 总奖励更差（ε-贪心导致频繁掉崖）
Expected SARSA 综合表现最好
